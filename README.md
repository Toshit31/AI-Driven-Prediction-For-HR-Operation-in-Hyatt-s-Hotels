# Employee Attrition Prediction (Hyatt Case)

This project uses machine learning to predict employee attrition and support proactive HR decision-making.

## Features
- Predict attrition risk
- Probability-based output
- Streamlit UI

## How to Run

pip install -r requirements.txt
streamlit run app.py

## Dataset
IBM HR Analytics Dataset
